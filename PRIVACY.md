# Политика конфиденциальности Form Saver Pro

**Дата вступления в силу:** 2026-03-29

## 1. Сбор данных

**Form Saver Pro НЕ собирает никакие данные пользователей.**

Расширение работает полностью локально в вашем браузере. Мы не:
- Собираем персональные данные
- Отправляем данные на серверы разработчика
- Используем аналитику или трекеры
- Передаём данные третьим лицам

## 2. Хранение данных

Все данные хранятся **локально** в браузере:
- Сохранённые формы
- Настройки расширения
- Мастер-пароль (в хэшированном виде)
- История действий

Данные хранятся в `browser.storage.local` и доступны только вашему браузеру.

## 3. Шифрование

При включённой опции шифрования:
- Данные шифруются алгоритмом AES-GCM 256-bit
- Ключ деривируется из пароля через PBKDF2 (100,000 итераций)
- Пароль не хранится нигде
- Без пароля данные невозможно восстановить

## 4. Разрешения

Расширение запрашивает следующие разрешения:

| Разрешение | Зачем нужно |
|------------|-------------|
| `storage` | Хранение данных форм локально |
| `tabs` | Определение активной вкладки для восстановления форм |
| `activeTab` | Доступ к текущей вкладке для работы с формами |
| `contextMenus` | Контекстное меню для быстрого доступа |
| `commands` | Горячие клавиши |
| `notifications` | Уведомления о сохранении/восстановлении |
| `alarms` | Автоочистка старых данных, напоминания |
| `<all_urls>` | Работа с формами на любых сайтах |

## 5. Сторонние сервисы

Расширение **не использует** сторонние сервисы:
- Нет внешних API
- Нет аналитики (Google Analytics и др.)
- Нет трекеров
- Нет рекламы

Единственное внешнее подключение:
- Загрузка фавиконок сайтов: `https://www.google.com/s2/favicons` (опционально, для отображения иконок сайтов в списке форм)

## 6. Обновления

При обновлениях расширения:
- Данные пользователя сохраняются
- Может потребоваться миграция формата данных
- Критические обновления безопасности устанавливаются автоматически

## 7. Безопасность

Рекомендации по безопасности:
- Используйте мастер-пароль для защиты
- Включайте шифрование для чувствительных данных
- Регулярно делайте резервные копии (экспорт)
- Не сообщайте пароль никому

## 8. Дети

Расширение не предназначено для детей младше 16 лет без надзора взрослых.

## 9. Изменения политики

Мы можем обновлять политику конфиденциальности. Актуальная версия всегда доступна в репозитории расширения.
